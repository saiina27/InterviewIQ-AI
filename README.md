# 🎤 InterviewIQ AI

An AI-powered Interview Preparation and Assessment System built with Python and Streamlit.

---

## 🚀 Features

- 🎯 Role-based interview questions (Beginner / Intermediate / Advanced)
- 🎤 Voice-to-text answer input
- 📊 Keyword + NLP-based answer scoring
- 🤖 AI feedback for each answer
- 📄 Resume parsing (PDF upload)
- 🎯 Resume skill matching system
- 🎥 Webcam integration for monitoring
- 📈 Performance dashboard (Plotly graphs)

---

## 🧠 Tech Stack

- Python
- Streamlit
- OpenCV
- SpeechRecognition
- Sentence Transformers (NLP)
- Plotly
- PyPDF / Resume Parsing
